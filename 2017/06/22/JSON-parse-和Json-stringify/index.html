JSON.parse()

parse是用一个字符串中解析出json对象，如：
var str=’{“name”:”alice”,”age”:”23”}’
结果：
JSON.parse(str)
Object
age:”23”
name:”alice”
proto:Object

注意：单引号写在{}外面，每个属性名必须用双引号，否则会报错

JSON.stringify()

stringify是从对象解析出字符串，如：
var a={a:1,b:2}
JSON.stringify(a)
‘{“a”:1,”b”:2}’
